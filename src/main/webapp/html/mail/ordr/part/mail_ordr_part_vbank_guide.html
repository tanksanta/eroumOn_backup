((dpstTermDtYear))-((dpstTermDtMonth))-((dpstTermDtDay)) ((dpstTermDtHour)):((dpstTermDtMinute)) 까지

주문일로부터 3일 이내 입금이 되지 않은 경우, 주문은 자동으로 취소됩니다.
(예시. 1월 12일 주문 후, 1월 15일 23시 59분까지 미 입금 시 자동 주문취소)
가상 계좌와 입금 금액이 다를 경우 정상적으로 주문 처리가 되지 않으니 반드시 확인 후 입금 바랍니다.
가상 계좌는 해당 주문에만 발급되는 일회성 계좌번호로 주문자명과 입금자명이 다르더라도 입금이 가능합니다.
입금 시 계좌번호 입력에 유의바랍니다.
주문은 입금 완료 기준으로 처리되며, 입금 전 상품이 품절되는 경우 배송이 불가하오니 상품 주문 시 유의 부탁드립니다.
